Finding Additional JDBC Driver Information
[This topic is pre-release documentation and is subject to change in future releases. Blank topics are included as placeholders.]

For more information about the Microsoft SQL Server 2005 JDBC Driver and SQL Server development in general, see the following online resources:

Resource Description

Microsoft SQL Server JDBC Driver Developer Center

This site provides documentation, technical articles, sample code, and other resources for the JDBC driver.

Data Access and Storage Developer Center

This site provides documentation, technical articles, sample code, and other resources for all data access technologies at Microsoft.

SQL Server Data Access Forum

This site serves as a community forum for data access to SQL Server by using SQL Server Native Client, OLE DB, ODBC, ADO, MDAC, JDBC, or SOAP/HTTP.

Microsoft Data Blog

This blog provides information about data access technologies and lets you interact directly with members of the Microsoft Data Programmability product team.

JDBC Newsgroup

This newsgroup is used to provide and discuss information involving the JDBC driver.

SQL Server Developer Center

This site provides technical articles, sample code, and other resources for SQL Server development.

Microsoft Knowledge Base

The articles in this database provide specific solutions, workarounds, and important information about the JDBC driver.

Microsoft Connect

This site allows you to search for driver issues submitted by customers, and it allows you to submit your own.
