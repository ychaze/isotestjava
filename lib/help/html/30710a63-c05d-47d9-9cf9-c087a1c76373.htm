SQLServerCallableStatement Class
[This topic is pre-release documentation and is subject to change in future releases. Blank topics are included as placeholders.]

Represents JDBC callable statements.

Package: com.microsoft.sqlserver.jdbc

Extends: SQLServerPreparedStatement

Implements: java.sql.CallableStatement

Syntax

public final class SQLServerCallableStatement

Remarks

The SQLServerCallableStatement class lets you specify the stored procedure name to call along with input and output parameters. This class also provides the ability to retrieve the return status value with the ? = call( ?, ..) syntax.
