executeUpdate Method (java.lang.String, java.lang.String[])
[This topic is pre-release documentation and is subject to change in future releases. Blank topics are included as placeholders.]

Runs the given SQL statement and signals Microsoft SQL Server 2005 JDBC Driver that the auto-generated keys indicated in the given array should be made available for retrieval.

Syntax

public final int executeUpdate(java.lang.String sql,
                               java.lang.String[] columnNames)

Parameters

sql

A String that contains an SQL statement.

columnNames

An array of Strings that indicates which column names of the auto-generated keys should be made available.

Return Value

An int that indicates the number of rows affected, 0 if using a DDL statement.

Exceptions

SQLServerException

Remarks

This executeUpdate method is specified by the executeUpdate method in the java.sql.Statement interface.
